SEO

Prácticas para mejorar la visibilidad en buscadores usando estructura, contenido y claridad técnica.

  • Jerarquía de títulos (H1/H2).
  • Meta description informativa.
  • Contenido que responda preguntas reales.

Hosting

Servicio donde se alojan los archivos de tu web para que sea accesible desde internet.

  1. Subes tu proyecto.
  2. Obtienes una URL.
  3. Tu web queda pública.
Qué es un servidor web (MDN)

Dominio

Es el nombre del sitio (por ejemplo, tusitio.com) que se conecta con el hosting.

Qué es un dominio (Cloudflare)

Lead

Persona que muestra interés y deja un dato de contacto (formulario, WhatsApp, llamada o correo).

Ejemplo: alguien que solicita una cotización desde la página.

CDN

Red de servidores que acelera la entrega de contenido al usuario por cercanía geográfica.

Qué es un CDN (Cloudflare)

HTTPS/SSL

HTTPS protege la comunicación entre el usuario y el sitio, y reduce riesgos de interceptación.
